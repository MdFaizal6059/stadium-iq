# 3-minute demo script

**00:00 · Landing.** Position the product: "WorldCupIQ AI is a Stadium
Decision Intelligence Operating System for FIFA World Cup 2026. Not a
chatbot — an operating system for decisions."

**00:30 · Executive Command Center.** Show the KPI wall, the flow chart,
the live risk feed. Call out one risk item.

**01:00 · AI Command Center.** Pick the preset "Gate E crowd density is
approaching threshold — what should we do?" Submit. As the request runs,
narrate the coordinator → specialist agents → decision engine flow.

**01:45 · Decision card.** Point out the seven-part decision: situation,
risks, predictions, options, recommended action, impact, confidence.

**02:15 · Crowd + Transportation intelligence.** Two zones + mode split.

**02:35 · Accessibility.** Verified step-free routes, assisted services.

**02:50 · API Status.** Show all five required keys and the responsible-AI
footer.

**03:00 · Close.** "Grounded on tournament knowledge. Confidence-scored.
Ready for Cloud Run."
